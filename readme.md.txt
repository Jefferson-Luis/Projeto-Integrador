PI – Online – Criar um menu responsivo
Continuando com o desenvolvimento do seu projeto, acrescente o efeito de menu responsivo em uma das suas páginas criadas.  

O menu com tele Desktop deve aparecer no formato normal e na tela mobile deve aparecer com o botão “sanduíche”, que possuí três linhas horizontais empilhadas umas sobre as outras  quando o tamanho da tela for abaixo  de 768 pixels.

Obs: Pode ser feito utilizando, Bootstrap, JQuery ou Javascript com CSS.  

  

Critérios de Avaliação  

Menu funcionando de forma responsiva  

Design da página desenvolvida(HTML e CSS)  

Organização das pastas e arquivos(HTML,CSS, imagens e etc) mostrado na UC7.  

  

Forma de Entrega  

1 arquivo HTML, 1 arquivo JS, 1 arquivo CSS – Os arquivos do seu projeto que consiga mostrar o efeito de menu responsivo! Todos em um arquivo zipado.
